## 인트아이 2018 해커톤 프론트 엔드 프로젝트

### 설치법
```
git clone https://github.com/Azurepeal/2018_INTI_HT_FRONTEND.git
cd 2018_INTI_HT_FRONTEND
sudo npm install

```
```
cd config
vi config.json
```
다음 JSON파일 작성
```
{
	targetURL: 'localhost'
}
```


### 실행법
```
sudo npm run dev
```

### 빌드법
```
sudo npm run build
```
빌드 후 dist 파일을 백앤드에서 렌더링 해주면 끝
